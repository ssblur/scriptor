# Casting Crystals

Casting Crystals are a tool used to change the **focus** of a spell.
This changes the effective self target of a spell, as well as altering the effect of some spells that also have
effects on their caster, such as when an item is deposited into an inventory.

A lectern will use a casting crystal's focus if the lectern is used with the crystal in hand.
<item href="scriptor:casting_crystal"></item>
## Player-Focused Crystals
<item href="scriptor:player_casting_crystal"></item>
A player-focused casting crystal can be used to focus on a specific player.

This will only work if the player is within 16 blocks of the casting lectern, 
and only on the specific player attuned to the crystal.

An unbound crystal can be attuned to yourself by using it.

A bound crystal can be unbound by crafting it by itself.
## Coordinate-Focused Crystals
<item href="scriptor:coordinate_casting_crystal"></item>
A coordinate-focused casting crystal can be used to focus on a block or a set of blocks.

This will only work for blocks within 16 blocks of the casting lectern.
If the block is an inventory, the inventory may be targeted.

An unbound crystal can be attuned to a new coordinate by using it on a block,
and can be attuned to up to 4 positions at once.
When used on a block, the crystal will target the inside face of the block,
so i.e. clicking on the top of a block will target the bottom face of the block above it.

A bound crystal can be unbound by crafting it by itself.
## Animal-Focused Crystals
<item href="scriptor:animal_casting_crystal"></item>

An animal-focused crystal is automatically attuned to all animals within 16 blocks of the casting lectern.

This focus cannot be changed or unbound.
## Mob-Focused Crystals
<item href="scriptor:mob_casting_crystal"></item>

A mob-focused crystal is automatically attuned to *all* mobs within 16 blocks of the casting lectern.

This focus cannot be changed or unbound.
