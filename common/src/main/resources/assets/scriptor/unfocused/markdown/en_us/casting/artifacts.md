# Artifacts

Artifacts are a rare sort of thing, I've only ever actually seen one.

Apparently this one was found in a chest deep within an ancient city, but I've heard they can
be found in all sorts of crevices, squirreled away by their previous owners.
I wouldn't be surprised if some folks had even put some in vaults or shrines, where they couldn't be 
lost, or worse, taken by an enemy.

These things hold some mighty powerful spells, though the one I have seems pretty simple, if devastating.
It contains an explosion powerful enough to level a house or two in one go, a spell I couldn't hope to cast
on my own.

Notable, too, is that when I cast with it, I did not feel drained as I usually do after casting a spell, but
I still couldn't wield it again right away. 
Maybe this is some older sort of magic that didn't use the caster's own reservoir of power?

Either way, I've been unable to produce one of these myself, and I can't even imagine the sorts of technology
you'd need to be able to fashion one like this nowadays.

If you find any, be sure to examine it closely. 
This artifact looked like an orb, but when I looked inside it I saw words that turned out to be a spell!
