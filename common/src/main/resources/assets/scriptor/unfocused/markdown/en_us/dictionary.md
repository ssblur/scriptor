# Nominomist's Guide
Any fledgling nominomists reading this tome: 

Unfortunately, not much is known about the magics described in this book.
I've tried to collate some basic information that might be helpful for learning and casting magic.

I'm certain, however, that there's more to be known, so I've included blank pages in the back 
of this book to be filled out at a later date.
Please do not hesitate to include your own notes using these pages.
