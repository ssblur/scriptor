# Engravings

Scattered around the world, I've occasionally seen words etched into stone or written on the ground.
These engravings, as I've called them, are actually magical in nature! 

It seems when you use the first word of one of these engraved spells, it will proceed along a chain and
finally cast the spell that was engraved when it reaches the last word.

Better yet, I've been able to replicate these engravings on my own, though less permanently!

<item href="scriptor:chalk"></item>

This chalk can be assigned a name in an anvil, then used to write that name on the ground.
By writing otherwise ordinary spells with it, I was able to replicate the spell-casting abilities of engravings.

If one were to branch one of these written spells, multiple different spells could even be cast at once by
using the first word in the chain, before the split. 
It seems each spell is cast at the end point of the written chain, just as if they were each written in the
same places individually.

Not to mention that these chalk spells could be much more powerful than anything I could cast on my own!
I was able to use just some chalk and ingenuity to stop a rainstorm! 
These "chalk circles" are truly miraculous things!

I wonder if there's a way to write a spell on the ground with magic itself...
