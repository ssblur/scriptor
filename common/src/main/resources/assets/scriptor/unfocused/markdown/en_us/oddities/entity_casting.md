# Observations

Some mobs can apparently cast magic all their own, much like I can!

I've decided to include an "Observations" section in this journal, so that if I hear one of them casting a spell
again, I can jot it down for later.

So far, I've only heard Evokers and Illusioners cast spells.
I'm not sure, but there may be more casters like them out there. 
If so, I want to find them and stick around long enough to hear what sort of spells they cast!
